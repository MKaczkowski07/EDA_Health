# EDA.Health

**Online Report:** https://mkaczkowski07.github.io/EDA_Health/

Exploratory Data Analysis (EDA) of health, nutrition, and physical activity indicators,
with a focus on obesity and lifestyle-related risk factors.

## Project objective
The goal of this project is to explore relationships between:
- diet (fruit & vegetable consumption),
- physical activity,
- socio-economic factors,
and obesity prevalence.

## Data
- Source: Behavioral Risk Factor Surveillance System (BRFSS)

- Unit of analysis: U.S. states + District of Columbia

- R

## How to run
1. Open `EDA_1.Rproj` in RStudio
2. Open `EDA_RMD.Rmd`
3. Knit to HTML

## Output
The main output is an HTML report (`EDA_RMD.html`) containing:
- descriptive statistics,
- visualizations,
- interpretation of results.
